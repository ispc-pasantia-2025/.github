# 📋 Descripción del Proyecto

El presente proyecto tuvo como finalidad la actualización, reorganización y optimización integral del sitio web institucional de la Dirección General de Educación Superior (DGES), con el propósito de mejorar la accesibilidad a la información, modernizar la experiencia de navegación y fortalecer la identidad visual institucional.

Como parte del proceso, se llevó adelante un rediseño completo de la interfaz, acompañado de un rebranding de los logos institucionales y una reorganización masiva del contenido existente. Además, se desarrolló una estructura responsive adaptada a múltiples dispositivos, incorporando navegación accesible mediante teclado y un menú hamburguesa optimizado para dispositivos móviles.

La intervención incluyó la reestructuración de secciones clave como noticias, concursos y documentos oficiales, priorizando la jerarquización visual, la claridad del contenido y la mejora de la experiencia de usuario.

# 🚀 Funcionalidades desarrolladas

## 📰 Slider Hero Dinámico y seccion de Noticias

Para optimizar la experiencia de navegación, diseñamos un plugin propio que gestiona integralmente la sección de noticias mediante tecnología AJAX. El componente incluye un slider dinámico para la página de inicio y una interfaz de tarjetas interactivas que despliegan el contenido completo de forma dinámica. Al evitar la recarga total del sitio para visualizar cada noticia, se logró reducir drásticamente los tiempos de espera y mejorar la fluidez visual, ofreciendo una navegación continua tanto desde la home como en la vista individual de artículos.

---

## ⚡ Accesos Rápidos desde el home

Con el objetivo de modernizar la estética integral del sitio y optimizar la usabilidad, se implementó en la página de inicio un sistema de tarjetas interactivas para las secciones de mayor tráfico. Este módulo de accesos rápidos centraliza enlaces estratégicos —como el Campus Virtual, Revista EFI y el área de Gobierno— integrando iconografía institucional y tooltips informativos. Esta solución no solo facilita una navegación intuitiva hacia los recursos y plataformas complementarias, sino que transforma la experiencia del usuario mediante una interfaz dinámica y visualmente renovada.

---

## 🎓 Área “Academico”

Se desarrolló la sección estratégica 'Academico', orientada a estudiantes y aspirantes, la cual integra los bloques de Ingreso 2026, Carreras, Nuestros Institutos y el Sistema de Gestión de Estudiantes. 
Para para el area de Institutos y Carreras, desarrollamos un segundo plugin personalizado basado en AJAX que transforma la visualización de datos en tablas interactivas accesibles. Esta solución permite a los usuarios filtrar contenidos en tiempo real y organizar resultados mediante la interacción directa con los encabezados de las tablas, garantizando una búsqueda eficiente y una claridad visual superior frente a las estructuras estáticas convencionales.

---

## 👩‍🏫 Sección Docente

Se diseñó un espacio exclusivo para el personal docente que centraliza información crítica sobre el ingreso a la carrera, gestión de horas cátedra, cargos directivos, convocatorias y vacancias. El trabajo técnico se centró en la reorganizacion y optimizacion del flujo de navegación, logrando reducir significativamente los pasos necesarios para acceder a secciones prioritarias. Mediante la unificación de datos redundantes y una reestructuración de los menús, se optimizó la accesibilidad a los procesos administrativos, permitiendo que los docentes localicen la información esencial de manera más ágil y directa.

---

## 🧩 Area Equipo Técnico

Se implementó una arquitectura visual modular diseñada específicamente para las áreas técnicas y administrativas, integrando sectores clave como Alfabetización Digital, Apoyo Administrativo y Equipos de Gestión. Esta estructura modular permite una segmentación clara de la información técnica institucional, facilitando la identificación de cada área mediante una jerarquía visual limpia. Gracias a este enfoque de diseño, el sitio adquiere una alta escalabilidad, permitiendo la incorporación de nuevas dependencias administrativas en el futuro sin comprometer la integridad estética o funcional del sitio.

# 🛠️ Aspectos técnicos y prioridades

Durante el desarrollo del proyecto se priorizaron los siguientes ejes:

## Accesibilidad

- Implementación de criterios de accesibilidad AA/AAA.
- Navegación optimizada mediante teclado.
- Mejora del contraste visual y estructura semántica.

## Diseño Responsive

- Adaptación completa para dispositivos móviles, tablets y escritorio.
- Optimización de navegación y visualización multiplataforma.

## Organización del Contenido

- Reestructuración y priorización masiva de contenido institucional.
- Mejora en la arquitectura de la información.
- Jerarquización visual de secciones estratégicas.

## Escalabilidad y mantenimiento

- Componentización de secciones reutilizables.
- Organización modular del desarrollo.
- Optimización para futuras actualizaciones institucionales.
