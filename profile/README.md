# Innovación y optimización en la aplicación web de la Dirección General de Educación Superior desde la práctica colaborativa

<br/>

## 📋 Descripción del Proyecto

El presente proyecto tiene como objetivo principal realizar un refactor visual integral del sitio web de la Dirección General de Educación Superior (DGES), con el propósito de revalorizar y jerarquizar el contenido institucional mediante la modernización de su interfaz y la implementación de un diseño responsive que se adapte eficientemente a múltiples dispositivos y plataformas digitales.

### 🎯 Objetivos Principales
- Modernizar la interfaz visual del sitio web institucional
- Implementar un diseño responsive para múltiples dispositivos
- Mejorar la experiencia de usuario (UX/UI)
- Jerarquizar y organizar el contenido de manera más efectiva
- Optimizar la accesibilidad y usabilidad del sitio

### 🎯 Objetivos Secundarios
- Implementar mejores prácticas de desarrollo web
- Documentar el proceso de rediseño
- Crear un sistema de gestión de contenidos eficiente
- Establecer una arquitectura escalable y mantenible
